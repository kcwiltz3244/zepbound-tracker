# Recovered data included

This build contains the recovered My Zepbound Journey data supplied by the user.

On first launch it imports these keys:

- mzjV9Grocery
- zepboundProcessDailyV1
- mzjV8Mission
- mzjV7Weights
- mzjV8DayOne
- mzjV81Nutrition

Before import, the current values of those keys are saved locally under `mzjV13PreRecoveredImportSnapshot`.
The one-time migration marker is `mzjV13RecoveredDataImported_2026_07_24`.
